# Predictive-Insights
This project focuses on analyzing student performance data using Exploratory Data Analysis (EDA) and Inferential Data Analysis techniques. The aim is to identify key factors that influence academic outcomes based on features such as attendance rate, study hours, parental support, extracurricular activities, and previous grades.
